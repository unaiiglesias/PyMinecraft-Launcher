config.ini should be loaded at the beginning of __init__ whereas launch_data should be loaded at the end

config.ini
- Title of the application (for incognito, AKA "PyMinecraft Launcher", mode)
- icon to use
- theme
- language
- show terror easter egg
- launcher version
- day (of the month, number) when version cache was saved
	- vanilla
	- forge
	- modpack

launch_data.json (This will be passed to launch_manager)
- Username
- version type to launch
- version number
- subversion number
- link to modpack
- ram amount
- installation path
